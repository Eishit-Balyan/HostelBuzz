# **App Name**: HostelBuzz

## Core Features:

- Real-time Updates: Display real-time updates from the hostel community in a feed format.
- Post Creation: Allow users to create new posts with category tags and text content.
- Upvote/Downvote: Implement upvote and downvote buttons (or confirm/false) on each post.
- Category Filters: Enable filtering of posts by category (Laundry, Mess, Caf√©, General).
- Timestamps: Show timestamps on each post to indicate when the update was made.
- Profile Icons: Display small profile icons next to each post to show the user who posted it.
- Real-time Notifications: Provide real-time notifications for new posts and updates.

## Style Guidelines:

- Primary color: Light blue (#A0D2EB) to evoke a fresh and friendly feeling.
- Background color: Very light blue (#EBF4FA), near white, for a clean modern aesthetic.
- Accent color: Warm peach (#FFDAB9) for interactive elements like upvote/downvote buttons and the floating '+' button.
- Body and headline font: 'PT Sans', a humanist sans-serif that blends a modern look with approachability.
- Use simple, clear icons for category tags and interaction buttons.
- Design the feed layout similar to Reddit or other social media platforms, with clear separation between posts.
- Implement subtle animations for upvotes/downvotes and new post notifications.
